Temperature Conversion

This ruby program will prompt the user for a temperature in degrees Celsius and let the user know what the corresponding temperature is in Fahrenheit. 


def celsius_to_fahrenheit(temp_celsius)
  return temp_celsius * 9.0 / 5.0 + 32.0
end

puts "Enter degrees in Celsius: "
celsius = gets.chomp.to_f
puts "The temperature is #{celsius_to_fahrenheit(celsius)} degrees Fahrenheit"

def ordinal(number)

  ending_digit =  number % 10
  ending_two_digits = number % 100

  if ending_digit == 1 && ending_two_digits != 11
    return "that's the #{number}st item!"
  elsif ending_digit == 2 && ending_two_digits != 12
    return "that's the #{number}nd item!"
  elsif ending_digit == 3 && ending_two_digits != 13
    return "that's the #{number}rd item!"
  else
    return "that's the #{number}th item!"

  end
end

Foobar

This ruby program is an adaptation of a classic tech interview problem which displays sequences of the Foobar pattern. 


puts "How many items do you want to see? "

user_number = gets.to_i

def foobar(how_many_numbers)

  how_many_numbers.times do |count|

    count += 1

    foo = count % 3 == 0
    bar = count % 5 == 0

    if foo && bar
      puts "foobar"
    elsif foo
      puts "foo"
    elsif bar
      puts "bar"
    else
      puts count
    end

  end
end
